* Este proyecto forma parte del programa de formación Oracle Next Education (Alura Challenger 2). Originalmente desarrollado en Vanilla JS, ha sido **migrado y modernizado** a una arquitectura robusta basada en React 19.

[🚀 ¡Puedes probarlo aquí!](https://alejandrobh.github.io/one-hanged)

## Tecnologías Usadas 🛠️

* **React 19** (Hooks, Context, Web Components)
* **Vite** (Next Generation Frontend Tooling)
* **CSS3** (BEM Methodology, Variables, Flexbox/Grid)
* **Local Storage API** (Persistencia del progreso)

## Características ✨

* 🌗 **Modo Oscuro/Claro**: Interfaz adaptable con persistencia de tema.
* 🎮 **Múltiples Categorías**: Informática, Frutas, Animales, Videojuegos, y más.
* ✏️ **Palabras Personalizadas**: Agrega y gestiona tus propias palabras para jugar.
* ⌨️ **Interactividad**: Teclado virtual animado y lógica de juego optimizada.
* 📊 **Persistencia de Datos**: El puntaje máximo y tus palabras se guardan automáticamente.
* 📱 **Diseño Premium & Responsive**: Experiencia visual fluida en cualquier dispositivo.

## Instalación Local 💻

```bash
# Clonar el repositorio
git clone https://github.com/AlejandroBH/one-hanged.git

# Entrar al directorio
cd one-hanged

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```
